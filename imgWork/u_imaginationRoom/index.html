Client Personal Project

Imagination Room

Project
Experience Web application of Infusing Imagination

Job Role
Art direction, UI / UX Design, Visual Design & Web Development

Skills Used
Photoshop, Illustration, Flash & Action Script3
The Imagination Room is a virtual web space for the user to have an opportunity to transcend the real world and to experience bizarre imagination spaces. It has plenty of animations and interactions to keep the user interested and pleased.

All of the images, photograph illustrations, animations, and interactions in this project are used to help users to build on their creative life in the real world a little bit more.

  • Thesis Project, Web Design & New Media at Academy of Art University, 2011
