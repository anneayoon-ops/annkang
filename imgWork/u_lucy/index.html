Lucy Script

The interactive scripts for ‘Lucy’ was to help increase consumer sales of the Universal’s blu-ray release. The ‘Lucy’ script is a unique website designated towards taking the user on an immersive journey. There are 118 pages with animated interactions on every page and feature.
